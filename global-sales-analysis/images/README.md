# 📈 Project Visualizations

This folder contains the core visual assets generated during the **Sales Efficiency Analysis** project. 

These charts support the key strategic insights and data correlations presented in the main report.

## 📂 Visual Assets Directory

* 📉 `sales_trend_2010_2017.png` — Global sales dynamics showing the 2012 peak, systemic recession, and the anomalous growth of the Snacks category.
* 📊 `abc_analysis.png` — Profitability breakdown highlighting the 5 key categories that generate 80% of total profit.
* 📅 `weekday_anomalies.png` — Daily purchasing patterns revealing specific spikes (e.g., Friday Cosmetics spike).
* 🎯 `scatter_delivery_profit.png` — Scatter plot (R²=0.007) proving the lack of direct correlation between shipping time and total profit.
