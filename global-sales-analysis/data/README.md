## 📊 About the Dataset

This dataset contains sales data for a global company that sells products both online and offline.

The data is divided into three separate tables (CSV files):

* 📄 **`events.csv`** — Sales records covering a period of several years.
* 📄 **`products.csv`** — Product categories and their corresponding item codes.
* 📄 **`countries.csv`** — Countries, regions, and their respective location codes.
